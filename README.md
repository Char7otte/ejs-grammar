## Snippets

→ Denotes the `TAB` key.

| Snippet→ | Output |
| --- | --- |
| `ejs→`    | `<% %>` - No output tag         |
| `ejs=→`  | `<%= %>` - Outputs HTML value    |
| `ejs-→`  | `<%- %>` - Outputs unescaped     |
| `ejs--→`  | `include` statement             |
| `ejsforof→`  | `for...of` Javascript Loop   |
